# Search For Cheap Flight and Get Notified via Email or SMS
## 0. If you're new to the program try running:
- `pip install pipenv` if there is no pipenv in system
- `pipenv install`
- `python getting_started.py`

## 1. If you're a developer:
- get the required api values and stuff
- api used
  1. [sheety]('https://dashboard.sheety.co/')
  2. [Tequila Search Api](https://tequila.kiwi.com/portal/docs/tequila_api/search_api)
  3. [Tequila Location Api](https://tequila.kiwi.com/portal/docs/tequila_api/locations_api)
  4. [Twilio for sms](https://www.twilio.com/docs/usage/api)
## 2. To start using the app
- `python main.py`
